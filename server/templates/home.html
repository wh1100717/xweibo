WHAT WE DO

Friends

We can get:friends geographical distribution,the number of reposts and comments,the information of friends,the number of friends,etc.

Micro-Bo

We can get the number of micro-Bo released within a week by Sina API.Through word clustering method, compared with the hot words, get the matching degree;The curves of the number of the reposted-micro-Bo over time.

Influence

We use an improved algorithm based on pagerank algorithm to get people's influence in Social Network.
